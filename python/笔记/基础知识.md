## 3. 时间的表示（f_string 格式）
```python
import math
s, v = map(int, input().split())
time_needed = math.ceil(s / v) + 10  
target_time = 8 * 60  
start_time = target_time - time_needed
if start_time < 0:
    start_time += 24 * 60
hour = start_time // 60
minute = start_time % 60
print(f"{hour:02d}:{minute:02d}")
```

# 2.`input`

## 1.多行数据的读取

input只会读取一行，想要读取多行，就必须循环读取，注意，`input`只会读取字符串
```python
lines = []
print("请输入内容（输入空行结束）：")
while True:
    line = input()
    if line == "":  # 遇到空行结束
        break
    lines.append(line)
result = '\n'.join(lines)
print("输出结果：")
print(result)
```
或者也可以用`sys.stdin.read()`来读取

最简单的方法是用for分别运算每一行的数据

## 2.一行数据不同类型的输入

`split()`方法会分割成一个列表
```python
num, text = input().split()
num = int(num)

# 输入: "0 singer pop music"
parts = input().split(maxsplit=1)  # 只分割一次
num = int(parts[0])                 # 0
text = parts[1]                      # "singer pop music"
print(num, text)
```


# 3.`abs`函数(绝对值函数)

```python
# 整数
print(abs(-5))    # 5
print(abs(5))     # 5

# 浮点数
print(abs(-3.14)) # 3.14
print(abs(3.14))  # 3.14

# 复数（返回模长）
print(abs(3+4j))  # 5.0 (因为 √(3²+4²) = 5)  , j在python就是代表复数
```

# 4. 迭代器和解包

解包：把一个可迭代对象里的元素，依次赋值给多个变量（必须一一对应，不然报错）
迭代器：python中迭代器是一个可以“一个一个产生元素”的对象，提供了一种**顺序访问容器元素**的方式，而不需要暴露容器的内部结构。

 迭代器运行会返回一个对象，这个对象只有被解包的时候才能被使用，如map函数
```python
a, b = map(int, input().split()) #这时候python会自动执行解包操作赋值a, b,输入3个值会直接报错
 
temp = map(int, input().split())  # 这时候temp是一个迭代器对象，print(temp)会返回类似于这个<map object at 0x0000027A13C32E60>
a, b = temp  # 解包
print('%d %d' % (a, b))
```

同理，revsered函数也会返回一个迭代器，这个函数支持一切可索引的可迭代对象，可以反转过来
```python
num = list(reversed(input()))
print(num) # 这样就可以返回了
```

# 5. 切片

Python的切片是 `[start:stop)` 或 `[start:stop:step]` 形式，即包含起始索引 `start`，不包含结束索引 `stop`。

这个step也可以是 -1 ，意思就是倒着来 ，如果是`[::-1]`，其实就是把列表倒转
