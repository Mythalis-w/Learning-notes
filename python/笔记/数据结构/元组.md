元组可以看成是一种特殊的列表，与列表不同的元组一旦建立就不能改变。==需要注意，这里指的不能被修改指的是地址，任何企图修改元组的操作都会发生错误。==
改变元素值也会改变地址
元组的基本形式是以括号()括起来的数据元素，他可以通过序号来引用其中的元素。

```python
# 创建元组
tuple_a = ()
tuple_b = tuple()

#创建一个只有一个元素的元组
tuple_c = (1)
tuple_d = (1, )
print(f'c 的值为{tuple_c}，数据类型是{type(tuple_c)}')  #c的值是1,数据类型是int
print(f'c 的值为{tuple_d}，数据类型是{type(tuple_d)}')  #d的值是(1, ),类型是tuple
# 创建元组必须要由逗号

# 访问元组
# 访问元组中的元素  
tuple_e = (1, 2, 3, 4, 5)  
print(tuple_e)

# 访问元组中第一个元素  
print(tuple_e[0])  
print(tuple_e[-1])

# 访问多个元素  
print(tuple_e[1:3])  #输出(2, 3)
"""
```

因为不能修改元组指的是不能修改地址，但是对于可变元素是可以修改的

```python
tuple = (1, 2, 3, [1 ,2 ,3]) # 这个列表占了地址4，列表里面的元素是可以修改的
print(tuple[3][0])
tuple[3][0] = 4
print(tuple) # 变为了(1, 2, 3, [4, 2 ,3])
```
