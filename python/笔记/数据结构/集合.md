集合，就是指看上去无序，没有重复值的列表

```python
data = {245 ,7 ,27, 136, 8, -1}  # 或者，可以data = set()
print(data) # 输出的是{245, 7, 136, 8, 27, -1}
# 顺序并不一致，因为他是按照哈希表是输出的

print(len(data)) # len函数也能用于集合

# 集合中添加和删除元素
data.add(9) # 这个添加的位置不确定，只知道是添加了
data.remove(9) # 9就被删除了

# 没有重复值
data = {245 ,7 ,27, 136, 8, -1, 8 ,245, -1}
print(data) #输出依旧是{245, 7, 136, 8, 27, -1}

#判断值是否在集合中
data = {245 ,7 ,27, 136, 8, -1}
print(7 in data) #会返回True
```

除了这些增减删等操作，也包含的数学上的意义

```python
data = {245 ,7 ,27, 136, 8, -1}
data_2 = {245 ,7 ,27, 136, 8, -1, 8, -1}

print(data.intersection(data2))  # 交集
print(data.union(data2)) # 并集
print(data.difference(data2)) #集合的减法
```
