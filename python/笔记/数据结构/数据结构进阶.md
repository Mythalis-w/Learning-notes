# 切片

切片这个操作被广泛运用于数据结构中，列表、元组等都能切片
==注意：切片的写法是 \[start:stop], stop后面也可以加上 :step ，规则是左闭右开  ==

```python
# 列表的切片
list=[1, 2, 3, 4, 5]
print(list[1:3]) #输出 [2, 3] 
# 注意，列表切片后仍然是列表，不论切片切了几个元素

# 元组的切片
tuple=(1, 2, 3, 4, 5)
print(tuple[1:3]) #输出(2, 3) 切片后仍然是列表
```

这种方法适用于相同的“键”对应不同的“值”的时候，因为此时并不能构造键值对（键与值并不一一对应），自然也不能用键值对的items()方法，还有一些其他的构造方法

```python
# 一个键对应多个(时间, 值)对
data = {
    'temperature': [('09:00', 25.5), ('12:00', 28.0), ('15:00', 26.5)],
    'humidity': [('09:00', 65), ('12:00', 60), ('15:00', 70)]
}
# 一个键对应的值可以是一个列表，甚至可以是一个键值对，当然，比较常见的作法是，先套一个列表，再套一个键值对 

# 访问
for time, value in data['temperature']:
    print(f"{time}: {value}°C")
```

```python
# 当每个值有多个属性时,列表套字典
students = {
    'class1': [
        {'name': 'Alice', 'score': 85, 'age': 20},
        {'name': 'Bob', 'score': 78, 'age': 21}
    ],
    'class2': [
        {'name': 'Charlie', 'score': 92, 'age': 19}
    ]
}

# 访问 class1 的所有学生
print(students['class1'])  
# [{'name': 'Alice', 'score': 85, 'age': 20}, {'name': 'Bob', 'score': 78, 'age': 21}]

# 访问 class1 的第一个学生
print(students['class1'][0])  
# {'name': 'Alice', 'score': 85, 'age': 20}

# 访问 class1 的第一个学生的名字
print(students['class1'][0]['name'])  
# 'Alice'

# 访问 class1 的第一个学生的分数
print(students['class1'][0]['score'])  
# 85

# 访问的方法是逐层访问，下面会放出另一种访问方式
# 遍历所有班级
for class_name, student_list in students.items():
    print(f"班级: {class_name}")
    for student in student_list:
        print(f"学生:{student['name']},分数:{student['score']},年龄: {student['age']}")

#这个是遍历访问，
```
