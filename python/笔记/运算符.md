
逻辑运算符
有 `not and or in is`

- **not**：一元运算符，其规则是“非假即真”。
    
- **or**：两个参与运算的对象有一个为真则返回真，两个都为假才返回假。
    
- **and**：两个参与运算的对象有一个为假则返回假，两个都为真才返回真。
    
- **is 与 is not**：身份操作符，判断左右两侧对象是否为同一对象，相同返回 True，不同返回 False。
    
- **in 与 not in**：成员操作符，判断左侧对象是否存在于右侧对象中。

```python
# not 即真变为假,假变为真
print(not 1)  # False
print(not False) # True

# or 或   and 与

#is, is not
is_a=1
is_b=1 
print(is_a is is_b) # True
print(is_a is not is_b) # False

# in  列表和键值对都可以用in
in_a = 'a'
in_string = 'abcd'
print(in_a in in_string) # True  
```
