
# 1.字符串的表示

python中的字符串通常由单引号，双引号，三个单引号或三个双引号包围的一串字符组成。

字符串中的字符可以包含数字，字母，中文字符，特殊符号，以及一些不可见的控制字符，
如换行符，制表符等。例如以下都是字符串： ` 'abc' "abc" '''abc''' """abc"""`

这些表达方式本质上是等价的，但是有时候只能有一些，有时候需要转义符才可以使用
常见的转义字符串包括 `\n \t \r …`  代表 换行符，制表符，回车。 转义符只转义后面一个字符
转义字符可以在引号中直接使用

```python
# Python示例
str1 = 'Hello World'
str2 = "Hello World"
# 两个字符串完全相同
print(str1 == str2)  # True
#下面的是错的，因为单引号里面再出现单引号是错的，双引号同理
bad_string = 'It's a beautiful day' 
good_string = 'It\'s a beautiful day'  #这个就是对的，有转义符就可以，或者直接换双引号
```

三引号包括 `''' '''  """ """ ` 这种可以包括双引号和单引号。

## 原始字符串 （r）

在Python中，**在字符串前加 `r`** 表示这是一个"原始字符串"，其中的**反斜杠 `\` 不会被视为转义字符**

```python
# 普通字符串需要转义
path1 = 'C:\\Users\\Documents\\file.txt'
print(path1)  # C:\Users\Documents\file.txt

# 原始字符串无需转义
path2 = r'C:\Users\Documents\file.txt'
print(path2)  # C:\Users\Documents\file.txt

print(path1 == path2)  # True，两者是相同的
```

# 3. 格式化字符串

## 占位符

有 %s %d %f 等，%s 是字符串占位符，可以把任何东西强制转化为字符串输出，%d是整数型在占位符，如果是浮点数则取其整数部分，%f是浮点型占位符，可以输出小数，保留n位数则%.nf

```python
name = '梨花'
print('我的名字是%s' % name)

age = 12
print('我的名字是%s，今年%s岁' % (name , age )) # age是整型，也会被强制转化为字符串

number_int = 18
number _ float = 17.6
print('number_int的值为:%d,nunber_float的值为%d' % (number_int， number_float))
#会输出 18 17

print('number_int的值为:%d,nunber_float的值为%d' % (number_int， number_float))
print('number_int的值为:%d,nunber_float的值为%f' % (number_int， number_float))
print('number_int的值为:%d,nunber_float的值为%.2f' % (number_int， number_float))
# 分别输出 18 17 ；18 17.60000000 ； 18 17.60
```

注意，如果需要输出 `%`，这时候需要 `print('%d%%' % precent)` 这时候会输出 91% 而不是 91

还有一种格式化

```python
 num = 42

print(f"{num:05d}")   # 00042 (5位，补零)  (:15d 并不会补1)
print(f"{num:5d}")    # '   42' (5位，补空格，默认右对齐)
print(f"{num:.1f}")  # 保留1位小数
print(f"{num:<5d}")   # '42   ' (5位，左对齐)
print(f"{num:>5d}")   # '   42' (5位，右对齐)
print(f"{num:^5d}")   # ' 42  ' (5位，居中对齐)
print(f"{num:b}")     # 101010 (二进制)
print(f"{num:x}")     # 2a (十六进制)
```
时间的表示就可以02表示，2位补0
# 4. 字符串处理函数

- `string.capitalize()`  
    将字符串的第一个字母大写，其余字母小写
    
- `string.count()`  
    获得字符串中某一子字符串的数目
    
- `string.find()`  
    获得字符串中某一子字符串的起始位置，无则返回-1
    
- `string.isalnum()`  
    检测字符串是否仅包含数字与字母
    
- `string.isalpha()`  
    检测字符串是否仅包含字母
    
- `string.isdigit()`  
    检测字符串是否仅包含数字
    
- `string.islower()`  
    检测字符串是否仅包含字母小写
    
- `string.istitle()`  
    检测字符串中的单词是否首字母大写
    
- `string.isupper()`  
    检测字符串是否仅包含大写字母
    
- `string.join()`  
    拼接字符串
    具体用法：'分隔符'.join(==可迭代对象==)
```python
# '分隔符'.join(可迭代对象)
fruits = ['apple', 'banana', 'orange']
print('-'.join(fruits))   # apple-banana-orange

words_tuple = ('Hello', 'World')
result = ' '.join(words_tuple)
print(result)  # 输出：Hello World

# 连接字典的键
d = {'name': 'Alice', 'age': '25', 'city': 'Beijing'}
print(', '.join(d.keys()))  # name, age, city

# 连接字典的值
print(', '.join(d.values()))  # Alice, 25, Beijing
```
- `string.lower()`  
    将字符串全部转换为小写
    
- `string.split()`  
    分割字符串, ( )内填写需要分割的字符串 ' ' ,分割成几个就用几个变量去接受
	==**split这个函数会分割成列表**==，至于上面的用几个变量去接受，在分割和个数相同的时候python会执行解包操作 
    
- `string.swapcase()`  
    将字符串中大写字母转换为小写，小写字母转换为大写
    
- `string.title()`  
    将字符串中的单词首字母大写
    
- `string.upper()`  
    将字符串中全部字母转换为大写
    
- `len(string)`  
    获取字符串长度
	
- `string.replace()`
```python
text = "Hello\nWorld\n"
new_text = text.replace("\n", "")  # 删除所有换行符，括号内是 (旧内容,新内容)
print(new_text)  # HelloWorld

# 指定替换次数
text = "a,a,a,a,a"
new_text = text.replace("a", "b", 3)  # 只替换前3个
```
	
- `string.strip()`
	去除字符串**首尾**的空白字符或者指定字符（空格、换行符`\n`、制表符`\t`等）
```python
# 常见用法
s1 = "  12345  "
print(s1.strip())  # "12345"

s2 = "\n\n67890\n\n"
print(s2.strip())  # "67890"

s3 = "  \t  abc  \n  "
print(s3.strip())  # "abc"

# lstrip() - 只去掉左边的空白
"  hello  ".lstrip()  # "hello  "

# rstrip() - 只去掉右边的空白
"  hello  ".rstrip()  # "  hello"

# strip(指定字符) - 去掉指定字符
"---123---".strip('-')  # "123"
```
