# 1. 爬取千千音乐

搜索胡彦斌的音乐，刷新，拿到一些包，找到document，看**Request URL**发现有一些乱码（一个% 两个字母类的编码），这个称为URL encode，就是URL的编码，对应着某一个信息（我们搜索的关键字），我们可以找一个URL编码在线工具去看看是什么。
`https://tool.chinaz.com/Tools/urlencode.aspx`
发现 %E8%83%A1%E5%BD%A6%E6%96%8C 经过解码就是 胡彦斌 。

因此，对于 `https://music.91q.com/search?word=%E8%83%A1%E5%BD%A6%E6%96%8C` 中的`word=%E8%83%A1%E5%BD%A6%E6%96%8C` ，我们称word是键，URL编码是值 ，因此，我们可以把 ？前面的视为请求地址，？后面的视为参数，如果有两个参数，用&分割 ，也可以通过网页中的Payload去看看请求了什么参数

```python
import requests  
  
header = {'user-agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/128.0.0.0 Safari/537.36'}  
keyword = '胡彦斌'  
Response = requests.get('https://music.91q.com/search?word=%s' % keyword, headers=header)  
print(Response.text)
'''理论上传递URL，但是有时候传中文也可以，如上'''
```

# 2.爬取歌曲名字和链接

```python
import requests  
from bs4 import BeautifulSoup  
  
header = {'user-agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/128.0.0.0 Safari/537.36'}  
keyword = '胡彦斌'  
Response = requests.get('https://music.91q.com/search?word=%s' % keyword, headers=header)  
  
soup = BeautifulSoup(Response.text, 'html.parser')  
  
# 歌曲名字和链接  
music_name = soup.find_all(name='div', class_='song-box')  
for index in range(1, len(music_name)):  
    name = music_name[index].find_all(name='a', target='self')  
    print(name[0].string, 'https://music.91q.com' + name[0]['href'])  
  
  
# 歌手名字   主要是有两个人合唱的情况，所以这样写
singer = soup.find_all(name='div', class_='artist ellipsis')  
for i in singer:  
    singer_names = ''  
    singer_name = i.find_all(name='a')  
    for j in singer_name:  
        singer_names += j.string + ' '  
    print(singer_names)

	'''详情页的链接需要点开详情页抓包看看详情页的构成(最后一行的print的中间)'''
```
