


# 1.爬取图片

库的安装

需要安装requests这个库
`pip install requsets`
****
以 https://scrape.center/ 中的ssr1为例子，点进ssr1，打开开发者模式，选择network网页，刷新网页，就能看到我们抓取的包了

先爬取一个图片，选择你感兴趣的图片，点击它，并通过preview去看看是不是该图片，然后选择**Headers**，这里关注**General**下面的 **Request URL** 和 **Request Method** ,分别代表的我们需要请求的URL和方法，然后看 Request Headers 下面的数据，关注开头不是Sec的数据： **Referer 和 User-Agent**，因为服务器一般需要验证这两个代码是不是正确的,Referer是指的发送请求的网址，User-Agent则是用的浏览器称号等等，来验证你是不是一个浏览器发送的请求，然后就能编写代码了

```python
import requests  
  
header = {'referer':'https://ssr1.scrape.center/', 'User-agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/128.0.0.0 Safari/537.36'}  # 构造请求头，骗过服务器（ **Referer 和 User-Agent**）写成键值对

Response=requests.get('https://p0.meituan.net/movie/283292171619cdfd5b240c8fd093f1eb255670.jpg@464w_644h_1e_1c',headers=header) #构造请求，并用Request接受

'''
因为**Request Method**是get方法，所以我们用get，后面跟上我们请求的服务器的URL（**Request URL**），还有我们伪造的请求头
header
'''
print(Response)  #理论会回复200，表示正常
print(Response.content) # 表示打印我们爬取到的数据，因为是图片，所以print二进制数据就是.content
  
# 图片的本质是一堆二进制码，用二进制写一个图片就好了
with open('movie.jpg', 'wb') as f:  
    f.write(Response.content)
```

# 3. 抽取单页电影信息

库的安装

安装bs4提取网页数据的代码源自2
****
其中
```python
import requests  
from bs4 import BeautifulSoup  
  
  
header={'Referer':'https://scrape.center/', 'User-agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/128.0.0.0 Safari/537.36'}  
  
response=requests.get('https://ssr1.scrape.center/', headers=header)  
print(response.text)  

soup = BeautifulSoup(response.text, 'html.parser') #实例化，将soup作为解释器，'html.parser'是内置的解释器，不需要下载额外的包，而前面的就是我们要解析什么网页数据

'''
打开开发者工具的Elements的左上角，找到文字所在的html代码在哪里（用左上角的工具），找到后需要用代码定位这个标签，首先，标签名字是`div` , 类别(class)是 'p-h el-col el-col-24 el-col-xs-9 el-col-sm-13 el-col-md-16' ,这个标签下面是电影的所有文字信息，根据这两个线索就够了，也可以加别的
'''

result = soup.find_all(name='div', class_= 'p-h el-col el-col-24 el-col-xs-9 el-col-sm-13 el-col-md-16')  # 注意，class在python中是关键字，冲突了，所以我们要加一个下划线；然后我们用find_all这个函数去寻找相似点  
print(result) # 返回的是一个列表
print(len(result)) #返回10，这是因为这个页面的电影数量就是10个

#拿到电影的名字
print(result[0].h2.string) # !!h2是电影名字所在的标签，string是寻找里面的字符数据
# 等价于 print(result[0].find('h2').string)

# 拿到电影的标签 ，类似科幻 剧情 爱情等描述词
# 我们发现，这些词大都放在span标签中，如果我们还是按照上面这么用，他就只会返回一个值（因为它找到第一个满足span的东西就跳过了），因此，由于soup是这个文本的解释器，result继承了这个解释器，所以对列表元素也能用BeautifulSoup的方法，所以通过检索button标签就能拿到了,这样也可以避免拿到其他的span的数据
button = result[0].find_all(name = 'button', class_='el-button category el-button--primary el-button--mini')

for btn in button:
	print(btn.span.string)

# 然后我们获取其他的信息
info_list = result[0].find_all(name='div', class_='m-v-sm info')
print(info_list) #会出来两个，因为有两个地方满足数据

#----------------------------------------------------------------------------------
#以下为错误示例

for info in info_list:
	print(info.span.string)
	
'''
会打印
中国内地、中国香港
1993-07-26 上映
少了一个东西，这和上面注释的问题是一样的，只找到第一个满足的值，就走了
'''
#----------------------------------------------------------------------------------
# 下面的是正确的
for info in info_list:
	span_list = info.find_all(name='span')
	for s in span_list:
		if s.string == ' / ':
			pass
		else:
			print(s.string)
			
#获取电影分数
score = soup.find_all(name='p', class_='score m-t-md m-b-n-sm')
print(score[0].string.strip())
#strip是字符串函数，是吧字符串左右空格都搞掉
```

==可以这么理解，soup创建一个解释器，凡是调用这个解释器的子元素都可以继承这个功能，且find_all 返回的一定是一个列表==

# 4.获取整个网站信息

想要获取整个页面的数据，加一个循环就可以了
```python
import requests  
from bs4 import BeautifulSoup  
  
  
header={'Referer':'https://scrape.center/', 'User-agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/128.0.0.0 Safari/537.36'}  
  
response=requests.get('https://ssr1.scrape.center/', headers=header)  
# print(response.text)  
soup = BeautifulSoup(response.text, 'html.parser') #实例化，将soup作为解释器，'html.parser'是内置的解释器，不需要下载额外的包  
result = soup.find_all(name='div', class_= 'p-h el-col el-col-24 el-col-xs-9 el-col-sm-13 el-col-md-16')  
for i in range(0,len(result)):  
    print(result[i].h2.string)  
    button = result[i].find_all(name = 'button', class_='el-button category el-button--primary el-button--mini')  
  
    for btn in button:  
        print(btn.span.string)  
  
    info_list = result[i].find_all(name='div',class_='m-v-sm info')  
    for info in info_list:  
        span_list = info.find_all(name='span')  
        for s in span_list:  
            if s.string == ' / ':  
                pass  
            else:  
                print(s.string)  
  
    score = soup.find_all(name='p', class_='score m-t-md m-b-n-sm')  
    print(score[0].string.strip())  
    print("--"*10)
```
想要获取其他页面的，只要更改url就可以，比如第二页的url是`https://ssr1.scrape.center/page/2`
只需要改最后的2就可以了
所以想获取整个网站的10页的数据，只需要通过循环 格式化字符串更改url就可以了
```python
for page in (1,11):
	response=requests.get('https://ssr1.scrape.center/page/%d' % page, headers=header)
```
完整代码：
```python
import requests  
from bs4 import BeautifulSoup  
  
  
header={'Referer':'https://scrape.center/', 'User-agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/128.0.0.0 Safari/537.36'}  
for page in range(1,11):  
    response=requests.get('https://ssr1.scrape.center/page/%d' % page, headers=header)  
    # print(response.text)  
    soup = BeautifulSoup(response.text, 'html.parser') #实例化，将soup作为解释器，'html.parser'是内置的解释器，不需要下载额外的包  
    result = soup.find_all(name='div', class_= 'p-h el-col el-col-24 el-col-xs-9 el-col-sm-13 el-col-md-16')  
    for i in range(0,len(result)):  
        print(result[i].h2.string)  
        button = result[i].find_all(name = 'button', class_='el-button category el-button--primary el-button--mini')  
  
        for btn in button:  
            print(btn.span.string)  
  
        info_list = result[i].find_all(name='div',class_='m-v-sm info')  
        for info in info_list:  
            span_list = info.find_all(name='span')  
            for s in span_list:  
                if s.string == ' / ':  
                    pass  
                else:  
                    print(s.string)  
  
        score = soup.find_all(name='p', class_='score m-t-md m-b-n-sm')  
        print(score[0].string.strip())  
        print("--"*10)
```

# 5.存放在excle中
1.库的安装
我们需要pandas这个库

****
2.库的使用
构造出一个叫DataFrame的数据
```python
import pandas as pd

info = {
'姓名':['小王','小刘','小张'],
'性别':['男','女','男']
}
data = pd.DataFrame(info)
print(data)
```
输出一个excel表格
```python
   姓名 性别
0  小王  男
1  小刘  女
2  小张  男
```

下面是写入excel（需要openpyxl这个库）

```python
import pandas as pd

info = {
'姓名':['小王','小刘','小张'],
'性别':['男','女','男']
}
data = pd.DataFrame(info)
print(data)
data.to_excel('info.xlsx')
```

这样就会生成一个excel，如果不想要序号 `data.to_excel('info.xlsx', index = False)`
这样就好了，其中构造的列表的键是每一列的名字，值是一个列表，是这一列中每一行的数据
写入就慢慢写就可以了
