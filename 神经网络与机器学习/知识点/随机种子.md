```python
def set_seed(seed: int = 42) -> None:  
    random.seed(seed)          # 固定random库的随机种子  
    np.random.seed(seed)       # 固定numpy库的随机种子  
    torch.manual_seed(seed)    # 固定PyTorch的随机种子
```

我们对于每一个模型都要设置这三个种子，保证在random和numpy和PyTorch在随机的时候随机的结果是相同的，即 可复现

在后面调用这个函数就可以了，也可以通过循环搞多种子训练，最后取平均。
