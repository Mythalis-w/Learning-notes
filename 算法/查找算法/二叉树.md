我们关注的是位置编号（position）,结论编号（result）是我们的索引
### 步骤1：路径追踪（从根节点出发）

**规则：**

- `y`（是）→ 向左走 → position = position × 2`
    
- `n`（否）→ 向右走 → position = position × 2 + 1

### 步骤2：计算叶子节点起始编号

leaf_start = 1 << N = 2³ = 8   (除了1外共有三层，N=3)

### 步骤3：转换为结论编号

result = position - leaf_start + 1

通过这种算法，可以精准的定位到二叉树的结论编号，通过编号输出正确的结论
