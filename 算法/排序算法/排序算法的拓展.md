
# 1.结构体在排序的应用

如果在排序的同时想保留原索引，使用结构体排序就可以了，结构体定义一个索引和一个值，排序时直接交换结构体,总之，结构体在排序中可以很好的保留原排列的信息。

# 2.去重排序

去重前一般是要先排序

```C
int arr[1000]; //假设已经排序完成
int m=1; //m为去重后的数组个数,m=1因为第一个元素一定保留
for(int i=1;i<n;i++){
	if(arr[i-1] != arr[i]){
		arr[m++]=arr[i];
	}
}
for(int i=0;i<m;i++){
	printf("%d",arr[i]);
}
return 0;
```
