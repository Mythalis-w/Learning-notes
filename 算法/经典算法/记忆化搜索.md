这是一种能够缓存已遍历结果的办法

其基本代码为
```python
N = int(input())
ori_list = list(map(int, input().split()))
memo = [0] * (N + 1)  # 0表示未计算
def find_result(start):
    # 1. 命中缓存
    if memo[start] != 0:
        return memo[start]
    
    # 2. 记录路径
    path = []
    curr = start
    
    # 3. 搜索直到找到已知结果或匹配点
    while memo[curr] == 0:
        if curr == ori_list[curr-1]:  # 找到匹配点
            memo[curr] = curr
            break
        path.append(curr)
        curr = ori_list[curr-1]
    
    # 4. 确定最终结果
    target = memo[curr] if memo[curr] != 0 else curr
    
    # 5. 更新路径上所有点
    for node in path:
        memo[node] = target
    
    return target
```

拿一个题目举例子:
### Problem Statement

There are $N$ cells, cell $1,$ cell $2,\ldots,$ cell $N$, arranged in a line. Cell $i$ has an integer $A_i\ (i \le A_i \le N)$ written on it.

For each of $s=1,2,\ldots,N$, solve the following problem.

-   Initially, place a piece on cell $s$. After performing the operation "let $x$ be the integer written on the cell where the piece is placed, and then move the piece to cell $x$" $10^{100}$ times, output the number of the cell where the piece is placed.

其实就是寻找索引和值相同的结果，不相同就跳到与这个值相同的索引,直至相同位置

平常代码
```python
N = int(input())
ori_list = list(map(int, input().split()))


for index in range(1,N+1):
    c_index = index
    while True:
        if c_index == ori_list[c_index-1]:
            print(c_index,end=' ')
            break
        else:
            c_index = ori_list[c_index-1]
```

这个代码会在Atcoder中提交就会TLE，这时候就可以使用记忆化搜索

```python
N = int(input())
ori_list = list(map(int, input().split()))
memo = [0] * (N + 1)  # 创建缓存列表

def find_result(start):
    if memo[start] != 0: # 代表已经走过这个结果
        return memo[start]
    path = [] # 记录走过的路径
    curr = start
    while memo[curr] == 0:  # 循环如下程序，寻找最终结果和记录缓存
        if curr == ori_list[curr - 1]:
            memo[curr] = curr
            break
        path.append(curr)
        curr = ori_list[curr - 1]
    target = memo[curr] if memo[curr] != 0 else curr  # 如果遇到已缓存节点就直接读取（此时的curr=start，并没有动），如果经历过上面的程序，则memo[curr] = curr ，读取curr，这个程序也会在读取缓存不为0的时候结束，这是target读取memo[curr]
    for node in path:  
        memo[node] = target # 对每一个经历过的路径进行赋值

    return target

for i in range(1, N + 1):
    print(find_result(i), end=' ')
```
