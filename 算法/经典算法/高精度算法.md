高精度普遍用于一些大数字的情况，可以把每一位都存放在数组/列表里面，逐位运算

如高精度加法
```python
num_1 = [int(x) for x in reversed(input().strip())]  
num_2 = [int(x) for x in reversed(input().strip())]  
  
carry = 0  
result = []  
  
# 处理到较长数字的末尾  
for i in range(max(len(num_1), len(num_2))):  
    # 如果i超出了某个数的长度，就当作0处理  
    digit1 = num_1[i] if i < len(num_1) else 0  # 这个技巧对于处理越界的情况比较多见
    digit2 = num_2[i] if i < len(num_2) else 0  
  
    total = digit1 + digit2 + carry  
    result.append(total % 10)  
    carry = total // 10  
  
# 处理最后的进位  
if carry:  
    result.append(carry)  
  
# 输出  
print(''.join(map(str, reversed(result))))
```

高精度乘法
```python
def multiply_basic(a, b):  
    if a == "0" or b == "0":  
        return "0"  
  
    a_list = [int(x) for x in reversed(a)]  
    b_list = [int(x) for x in reversed(b)]  
    
    result = [0] * (len(a_list) + len(b_list)) # 结果列表 
    for i in range(len(a_list)):  
        for j in range(len(b_list)):  
            result[i + j] += a_list[i] * b_list[j] # 处理加和每一位  
    carry = 0  
    for i in range(len(result)):  #对每一位进位
        total = result[i] + carry  
        result[i] = total % 10  
        carry = total // 10  
  
    while len(result) > 1 and result[-1] == 0:  # 对于结果列表中对于的0删除  
        result.pop()  
    return ''.join(map(str, reversed(result)))  
  
  
a = (input())  
b = (input())  
print(multiply_basic(a, b))
```
