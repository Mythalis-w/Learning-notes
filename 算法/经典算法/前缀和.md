
前缀和就是把前n个（n从0到n）数字加起来并组合成一个数组叫做前缀和


求数组的最大子数组和的长度
输入`[3,-4,2,-1,2,6,-5,4]`

```python
num = list(map(int, input().split()))
n = len(num)
prefix = [0] * (n + 1)  # 多一个位置，prefix[0]=0
for i in range(1, n + 1):
    prefix[i] = prefix[i-1] + num[i-1]
    
min_prefix = prefix[0]  # 初始化为第一个前缀和
max_sum = float('-inf')  # 这个数负无穷

for i in range(1, n + 1):
    max_sum = max(max_sum, prefix[i] - min_prefix)  # 这个for循环包含了max在min后面的约束
    min_prefix = min(min_prefix, prefix[i])
    
print(max_sum)
```

对于 **子数组和为K的问题**  **最大/最小子数组问题**  都可以使用前缀和来解决
