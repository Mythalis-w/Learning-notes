一般用于删除某种元素的地方，我们拿一个题目来做例子
****
- 从键盘读入一个字符串到数组str中，再读入待删除的字符数n 及从键盘读入的n个用空格分隔的字符存放在字符数组cr中。调用自定义函数delet()，删除str字符串中给定的字符。如果字符串中没有指定字符，则不处理。

```define
void delet(char str[],char cr[],int n); 
```

```C
void delet(char str[], char cr[], int n) {
    int i, j, k;
    int found;
    i = 0;
    j = 0;
    while (str[i] != '\0') {
        found = 0;
        for (k = 0; k < n; k++) {
            if (str[i] == cr[k]) {
                found = 1;  // 找到匹配，需要删除
                break;
            }
        }
        
        // 如果不需要删除，则保留该字符
        if (!found) {
            str[j] = str[i];
            j++;
        }
        
        i++;  // 继续处理下一个字符
    }
    
    // 在新字符串末尾添加结束符
    str[j] = '\0';
}
```

这是双指针算法的代码，我们思考一下，假设从`abcdefgLLikl` 中删除`a e L n` 这串代码究竟是如何工作的

i=0 j=0 需要删除，跳过if    a b c d e f g L L i k l     变为i=1 j=0
i=1 j=0 不需要删除，if不跳过    b b c d e f g L L i k l     变为i=2 j=1
i=2 j=1 不需要删除，if不跳过    b c c d e f g L L i k l     变为i=3 j=2
……
i=4 j=3 需要删除，跳过if    b c d d e f e f g L L i k l   变为i=5 j=3
i=5 j=3 不需要删除  b c d f e f g L L i k l    变为i=6 j=4
…………

总之，j所在的位置，就是我们最后要输出的，而i可以视作检索指针，j则是输出指针，如果需要删除，输出指针不动，检索指针后移，再赋值从而实现同一数组输出的方法，最后为了结束输出，添加结束符 '\0' ，来实现字符的终止。
****
总之，i和j就是两个指针，一个负责前方检索，一个负责后方输出，

问题陈述

>在AtCoder Country，有一条规则是“当卡车司机驾驶 $A$ 分钟或更长时间时，必须至少休息 $B$ 分钟。”

你得到一个长度为 $N$ 的字符串 $S$ ，由‘ a ’和‘ b ’以及正整数 $A$ 和 $B$ 组成。求满足下列所有条件的整数对 $(l,r)$ 的个数。

—— $1\leq l \leq r \leq N$

—从 $S$ 的 $l$ 第1个字符到 $r$ 第1个字符的子字符串中‘ a ’的个数大于等于 $A$ 。

—从 $S$ 的 $l$ 第1个字符到 $r$ 第1个字符的子字符串中‘ b ’的个数小于 $B$ 。

也是双指针
```python
N, A, B = map(int, input().split())  
S = input()  
  
left = 0  
count = 0  
  
while left < N:  
    a_cnt = 0  
    b_cnt = 0  
    # 用 right 指针扩展窗口  
    right = left  
  
    # 这个 while 循环会尝试所有以 left 开头的子串  
    while right < N:  
        # 更新计数  
        if S[right] == 'a':  
            a_cnt += 1  
        elif S[right] == 'b':  
            b_cnt += 1  
  
        # 检查条件  
        if a_cnt >= A and b_cnt < B:  
            count += 1  # 找到一个符合条件的子串  
            # 关键：这里不 break！继续扩展 right            right += 1  
            continue  
  
        # 如果 b 太多了，再怎么扩展 right 也没用了  
        if b_cnt >= B:  
            break  # 跳出内层循环，移动 left  
        # 如果 a 不够且 b 也不多，继续扩展  
        right += 1  
  
    # 以当前 left 开头的所有可能子串都检查完了  
    left += 1  # 移动左指针  
  
print(count)
```
