- `string.length()` or `string.size()`
	返回string的大小
	
- `string.substr(m, n)` `string.substr(m)`
	从指定index m开始，提取指定长度n的字符串，如果没有n，就直接提取到末尾
	
-  `string.find(str，begin,end)`
	寻找这个字符串中的子串，如果存在，则返回子串第一个字符的index，如果没有找到子串，则返回 -1 ，begin和end则表示从index：begin开始检索到end
```C++
#include<iostream>  
#include<string>  
using namespace std;  

int main() {  
    string s = "Hello, world!";  
    int temp = s.find("world");  
    cout << temp;  
    return 0;  
}
```
	
- `string.rfind(const string& str, size_t pos = npos)`
	这个函数和find一样，只不过是从后往前找，依旧返回第一个字符的index
	
- `string.insert(index,"XXX")`
	这个函数可以在index后面插入内容
	
- `string.erase(size_t pos = 0, size_t len = npos)`
	这个函数可以删除从index n开始，删除到index m结束
```C++
#include <iostream>
#include <string>
using namespace std;

int main() {
    string str = "Hello, World!";
    
    // 示例1：从位置7开始删除到末尾
    string s1 = str;
    s1.erase(7);  // 从索引7开始删除到末尾
    cout << "erase(7): " << s1 << endl;  // 输出: Hello, 
    
    // 示例2：从位置7开始删除5个字符
    string s2 = str;
    s2.erase(7, 5);  // 从索引7开始删除5个字符
    cout << "erase(7,5): " << s2 << endl;  // 输出: Hello, d!
    
    // 示例3：默认参数（从0开始删除到末尾）
    string s3 = str;
    s3.erase();  // 等价于 erase(0, npos)
    cout << "erase(): " << s3 << endl;  // 输出: (空字符串)
    
    return 0;
}
```
	
- `string.push_back(char c)`
	在字符串末尾追加一个字符
	
- `string.append(string)`
	在末尾追加一个字符串
	
- `string.pop_back()`
	删除最后一个字符
	
- `string.clear()`
	清除字符串的内容
	
- `string.empty()`
	检查字符串是否为空，返回true/false
