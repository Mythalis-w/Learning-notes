- `deque.push_back(ele)`  
    在双端队列**尾部**添加一个元素
    
- `deque.push_front(ele)`  
    在双端队列**头部**添加一个元素
    
- `deque.size()`  
    获取双端队列的大小（元素个数）
    
- `deque.pop_back()`  
    删除**最后一个**元素
    
- `deque.pop_front()`  
    删除**第一个**元素
    
- `deque.resize(num,elem)`  
    将双端队列的大小重新设置为 num（如果变长，新增元素为默认值；如果变短，多余元素被删除）
    
- `deque.insert(iterator pos, int count, ele)`  
    在位置 pos **前面**插入 count 个 ele 元素（count 可省略，省略则插入一个）
    
- `deque.erase(iterator start, iterator end)`  
    删除从 start 到 end 之间的元素（左闭右开区间），如果只传一个迭代器，则删除该位置的元素
    
- `deque.clear()`  
    删除所有元素，容器大小变为 0
