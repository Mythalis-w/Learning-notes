- `vector.push_back( )`
	这个方法是在一个向量后面添加数据
	
- `vector.size()`
	获取这个向量的大小
	
- `vector.pop_back()`
	删除最后一个数据
	
- `vector.resize(num,elem)`
	把这个向量的大小重新设置为num,（如果变长，新增元素为默认值；如果变短，多余元素被删除）
	
- `vector.insert(iterator pos, int count, ele)`
	这个的意思是在位置pos**前面**插入count个ele元素，注意pos的指向必须是一个迭代器
	并且count可以省略，省略后固定插入一个元素
	
- `vector.erase(iterator start, iterator end)`
	这个的意思是删除从start到end的之间的元素，里面的指向是迭代器（左闭右开）
	并且后面的end可以不写，则删除start指向的元素
	
- `vector.clear()`
	删除所有元素
