算法需要`#include<algorithm>`头文件

C++ 标准库提供了**全局的 `std::sort` 算法**，可以对一些STL容器进行排序

|       |                                                           |            |
| ----- | --------------------------------------------------------- | ---------- |
| 默认升序  | `std::sort(vec.begin(), vec.end());`                      | 从小到大排序     |
| 降序排序  | `std::sort(vec.begin(), vec.end(), std::greater<int>());` | 从大到小排序     |
| 自定义排序 | `std::sort(vec.begin(), vec.end(), cmp);`                 | 传入自定义比较函数  |
| 部分排序  | `std::sort(vec.begin(), vec.begin() + 3);`                | 只排序前 3 个元素 |
| 稳定排序  | `std::stable_sort(vec.begin(), vec.end());`               | 相等元素保持相对顺序 |

注意，list容器不能用全局的sort算法，但是他有自己的算法`list.sort()`
