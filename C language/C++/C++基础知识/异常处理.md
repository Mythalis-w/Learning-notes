
C++ 处理异常的机制由检查、抛出和捕获三部分组成，分别由三种语句来完成：
- try (检查)
- throw (抛出)
- catch (捕获)

其中正常的写法是try圈起容易出一场的代码，然后在try圈出的代码中去写throw，`throw` 不是凭空写的，它一定出现在**你能提前想到、且代码无法自动处理的错误场景**，最后catch去写在后面，用来处理这个异常（==catch和try都是同级的，必须并列去写==）


# 1.写法
try的写法很简单，`try{}`就可以,主要讲throw和catch

1. catch的常规写法就是
```c++
// 基础格式 
catch (异常类型 [变量名]) { 
	// 异常处理代码 
}
```
其中，里面的异常类型是什么都可以，变量名可以不写，也可以写
```c++
// 1. 捕获 int 类型的异常
catch (int e) {
    cout << "错误代码：" << e << endl; // e的值是throw后面的值
}

// 也可以不写变量名，只捕获类型（只知道类型，不使用异常值）
catch (int) {
    cout << "发生了int类型的错误！" << endl;
}
```

2. throw的常规写法是
```C++
// 1. 先定义错误类型
enum Error {
    UNDERFLOW,
    OVERFLOW,
    DIV_ZERO
};

// 2. 抛异常（最标准写法）
if (栈空)
    throw UNDERFLOW;

if (栈满)
    throw OVERFLOW;

// 3. 捕获
catch (Error e) {
    if (e == UNDERFLOW) ...
}
```
这个例子就是用了enum去定义Error类型，然后来处理异常

3. catch的特性
```C++
#include<iostream>
using namespace std;
int func(int x){
    throw x;
}

int main(){
    try{
        func(5);
        cout << "No Here" << endl; //这一行永远不会输出
    }
    catch (double) {
        cout << "异常发生，双精度型！" << endl;
    }
    catch (...) {
        cout << "任意类型异常！" << endl;
    }
    catch (int){
	    cout << "int异常" << endl;
    }
    cout << "End!" << endl;
    return 0;
}
```
很明显，这个异常满足任意类型和int类型的catch，然后根据运行顺序，只会运行第二个catch，然后就结束，第三个不会运行

# 2. 异常处理必要的说明

- 容易引起异常的语句或程序段必须放在 try 块中；如果在执行 try 块内的复合语句过程中发生异常，throw 抛出的异常信息传到 try_catch 结构，系统寻找与之匹配的 catch 子句；

- 如果在执行 try 块内的复合语句过程中没有发生异常，则 catch 子句不起作用，流程转到 catch 子句后面的语句继续执行；

- 一个 try_catch 结构中只能有一个 try，但却可以有多个 catch 块，以便与不同的异常信息匹配；

- 如果在 catch 子句中没有指定异常信息的类型，而用了三点删节号 “…”，则表示它可以捕获任何类型的异常信息；

- throw 语句中可以不包括表达式，将当前异常再次抛出给上一层的 catch 块处理；

- C++ 中，一旦抛出一个异常，而程序不捕获的话，那么系统就会调用一个系统函数 terminate，由它调用 abort 终止程序。

我们着重说下最后两条

对于倒数第二条，我们举个例子
```C++
#include <iostream>
using namespace std;

void func() {
    throw 100; // 第一层抛出异常
}

int main() {
    try {
        try {
            func();
        }
        // 内层 catch 接住异常
        catch (int e) {
            cout << "内层捕获到异常：" << e << endl;
            throw; // 关键！不带表达式，把这个异常再抛出去
        }
    }
    // 外层 catch 接住再抛出来的异常
    catch (int e) {
        cout << "外层也捕获到了：" << e << endl;
    }
    return 0;
}
```

首先注意，每一个不同级的catch必须对应一个同级的try，所以这里写两个try，具体意思去看代码里的注释，并不是一个很难理解的事情
