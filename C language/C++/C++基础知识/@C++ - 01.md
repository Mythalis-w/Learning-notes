
# 1.引用变量

C++的引用变量可以看作变量的别名，并且==一旦初始化就不能改变引用对象，且必须在一开始就被定义，不能后续定义==
```C++
int a = 10;
int &ref = a;  // ref 是 a 的引用（别名）
ref = 20;      // 修改 ref 就是修改 a
cout << a;     // 输出 20

int b = 30;
ref = b;       // 这不是让 ref 引用 b，而是将 b 的值赋给 a！
// ref 仍然引用 a，a 的值变成了 30
```

# 2.引用形参

先举C++用C中的指针的例子，来实现地址的传递

```C++
#include<iostream>  
using namespace std;  
  
void swap(int *a , int *b){  
    int t = *a;  
    *a = *b;  
    *b = t;  
}  
  
int main(){  
    int a,b;  
    cin >> a >> b;  
    cout << a << "\t" << b << endl;  
    swap(&a, &b);  
    cout << a << "\t" << b << endl;  
    return 0;  
}
```

引用是经过语法美化的指针，它让地址传递看起来像是值传递，但保留了直接修改原变量的能力。
==注意：引用不可以是临时值==

```C++
#include<iostream>  
using namespace std;  

// 改为引用形参版本
void swap(int &a, int &b){  // 注意：这里改为引用，参数前加 &
    int t = a;   // 直接使用 a，不需要解引用
    a = b;       // 直接修改原变量
    b = t;  
}  

int main(){  
    int a, b;  
    cin >> a >> b;  
    cout << a << "\t" << b << endl;  
    swap(a, b);  // 直接传变量，不需要取地址 &
    cout << a << "\t" << b << endl;  
    return 0;  
}
```

==&在定义变量的时候是引用，在表达式是取地址==

## `int &f(int &i)` vs `int f(int &i)` 的区别

| 对比项    | int &f(int &i)      | int f(int &i)      |
| ------ | ------------------- | ------------------ |
| 返回值类型  | **引用参数** (`int&`)   | **值** (`int`)      |
| 返回的是什么 | **返回i本身**（左值，有内存地址） | **返回i的副本**（右值，临时值） |
| 能否作为左值 | ✅ 可以（如 `f(x)=100`）  | ❌ 不可以              |
| 能否取地址  | ✅ `&f(x)` 可行        | ❌ `&f(x)` 不可行      |
| 内存开销   | 无拷贝                 | 有拷贝                |
| 典型用途   | 需要修改原对象、链式调用        | 只需要值               |
这里是一个比较难理解的地方，首先，我们讲下临时值和固定值：
- 临时值：一般指的没有固定的地址，只存在于表达式中，且表达式结束时销毁；
- 固定值：有地址，可在一定范围内存在。
```C++
void func() {
    int local = 10;        // 局部变量：有地址，不是临时值
    int temp = local + 5;   // local+5 是临时值（右值）
    
    &local;                 // ✅ 可以取地址
    // &(local + 5);        // ❌ 不能取地址，因为 local+5 是临时值
    
    int* p = &local;        // ✅ 指针指向局部变量
    // int* q = &(local+5); // ❌ 不能指向临时值
}
```

了解临时值和固定值的区别用，我们来了解**引用函数**，就是大标题的第一种表达
首先，==引用函数必须返回一个有地址的值，这个地址不能在函数结束被销毁（不能是函数的局部变量，但并不是不能是局部变量，而是返回的使用局部变量不能被销毁）,不能返回临时值（因为没有地址）==

```C++
int global = 10;

int& f(int &x) {
    x = x + 10;  // 计算结果存入全局变量
    int res = global+x;
    return x;    // 返回全局变量的引用
}  //像这个函数，返回x/global就是对的，返回res就是错误的，x是main函数用定义的，在main函数期间调用f这个引用函数，x的地址依旧存在，global是全局变量，地址始终存在，但是res的地址在f函数运行结束直接就被销毁了，所以返回res不行;如果传入改为int x，那么返回x也不对(因为函数还是引用函数，引用函数必须返回一个有地址的值，改了以后x变为了一个只在函数内部存在的值，所以不对)
```
引用函数可以直接赋值即其可以被当作左值
```C++
int &f1(int &i) {  // 返回引用
    i += 10;
    return i;      // 返回 i 本身
}
int f2(int &i) {   // 返回值
    i += 10;
    return i;      // 返回 i 的副本（临时值）
}
int main() {
    int x = 5;
    int y = 5;
    f1(x);  // x = 15
    f2(y);  // y = 15 
	
    // 2. 关键区别：返回值的使用
    f1(x) = 100;     // ✅ 可以！x 变成 100 ，因为f1(x)的返回值是x，相当于x=100，如果f1（x）的返回值是全局变量arr，则arr=100
    // f2(y) = 100;  // ❌ 错误！返回值不是左值
    
    // 3. 作为左值
    int &ref = f1(x);  // ✅ ref 是 x 的引用
    // int &ref2 = f2(y); // ❌ 不能将临时值绑定到普通引用
    int res = f2(y) // ✅
    
    // 4. 连续调用
    f1(f1(x));  // ✅ 可以，返回引用可以继续传参
    // f2(f2(y)); // ❌ 编译错误,因为f2的参数是引用参数，把&去掉就对的
    
    return 0;
}
```

# 3.函数的默认形参

在C++中，函数的形参可以有默认值`void print(char ch , int n = 1)`，这时候n有一个默认值，当函数不给n传参的时候，n就取默认值.

==注意：默认形参一律在非默认形参的右面,如`void print(int n = 1 , char ch)` 是错误的==

```C++
#include<iostream>  
using namespace std;  
  
void print(char ch , int n = 1){  
    for(int i=0;i<n;i++){  
        cout << ch;  
    }  
}  
  
int main(){  
    print('*');  
    cout << endl;  
    print('*', 3);  
    cout << endl;  
    return 0;  
}
```

==注意：`void fun(int a, int b=1, char c='a', float d=3.2);` 如果你不打算给b这个参数，那么c d你都不能给，不然会引起错误，如：`fun(2,  ,'c' ,3.14);` 直接报错，或者直接给b一个1，不改变b==

# 4.函数重载

C++中，允许同一作用域中有重名的函数,只要他们的形参不同，就是不同的（如果形参和函数名字都相同，不管返回值是否一致，那么这两个函数就是一样的）,因为C++是以函数签名来区分函数的,而函数签名由函数名和形参列表（包括类型和个数）组成的

如下
```C++
#include<iostream>  
using namespace std;  
  
int add(int a, int b){  
    return (a+b);  
}  
  
double add(double a, double b){  
    return (a+b);  
}  
  
int main(){  
    cout << add(3, 5) << endl;  
    cout << add(3.4 ,5.3) << endl;  
    return 0;  
}
```

# 5.函数模板

函数模板，也称为泛型算法，C++允许我们用函数模板去创造一个通用的函数类型
```C++
template<typename T>
T add(T x , T y){
	return x + y;
}
```

例子
```C++
#include<iostream>  
#include<string>  
using namespace std;  
  
template<typename T>  
T add(T x , T y){  
    return x+y;  
}  
  
int main(){  
    cout << add<int>(3 ,5) << endl;  
    cout << add<double>(3.1 ,5.5) << endl;  
    cout << add<string>("Hello", "World") << endl;  
    return 0;  
}
```

中间`add<int>`中间的int不用强制写，他会自动推导类型

# 6. 内存动态分配

C++的动态内存分配是通过new和delete来实现的，new是申请内存，delete是释放内存，new在创建的时候会返回一个指针，需要接收

```C++
#include<iostream>  
#include<vector>  
using namespace std;  
  
int main(){  
    int *p = new int;  //申请1个int的内存
    *p =3;  
    cout << p << " " << *p;  
    delete p;  //释放内存
    return 0;  
}
```

与此我们也可以申请多个内存来作为数组使用，同时对应的new和delete形式都会改变

```C++
#include<iostream>  
#include<vector>  
using namespace std;  
  
int main(){  
    int *p = new int[10];  
    for(int i=0;i<10;i++){  
        p[i] = 2*i+1;  
    }  
    for(int i=0;i<10;i++){  
        cout << p+i << " " << *(p+i) << endl;  //p[i]  
    }  
    delete[] p;  //释放内存  
    return 0;  
}
```
注意：`int *p = new int[10];`和`int *p = new int(10)`是不一样的，第一个是申请10个空间，第二个是申请一个空间并赋值为10.
## 析构函数

析构函数是C++中一个特殊的成员函数，在对象**被销毁时自动调用**，用于执行清理工作。

- 特性：
- - **无参数**，无返回值
    
- **不能重载**（每个类只有一个析构函数）
    
- 对象销毁时**自动调用**
    
- 主要用途：**释放资源**（堆内存、文件句柄、网络连接等）
