
- 函数模板是一个通用函数，其函数返回类型和形参类型不具体指定，用一个虚拟的类型来代表。函数模板提供了一类函数的抽象，这个抽象的通用函数就称为函数模板。

- 模板函数：在实际调用函数时，系统会根据实参的类型来取代函数模板中的虚拟类型从而实例化成具体的函数，这个由函数模板生成的具体函数称为模板函数。

# 函数模板

函数模板，也称为泛型算法，C++允许我们用函数模板去创造一个通用的函数类型
对于函数模板的声明
```C++
template <typename 类型参数>
返回类型 函数名(模板形参表)
{
    函数体;
}

template <class 类型参数>  // 早期版本用的是class
返回类型 函数名(模板形参表)
{
    函数体;
}
```

```C++
template<typename T>
T add(T x , T y){
	return x + y;
}
```

例子
```C++
#include<iostream>  
#include<string>  
using namespace std;  
  
template<typename T>  
T add(T x , T y){  
    return x+y;  
}  
  
int main(){  
    cout << add<int>(3 ,5) << endl;  
    cout << add<double>(3.1 ,5.5) << endl;  
    cout << add<string>("Hello", "World") << endl;  
    return 0;  
}
```

中间`add<int>`中间的int不用强制写，他会自动推导类型
当然，在定义的时候写两个参数的函数模板也是可以的 `template<typename T,typename A> `
## 有关函数模板的说明：

- 函数模板中允许使用多个类型参数，每个类型参数前必须有关键字`typename`或`class`；
- 在`template`语句与函数模板定义语句之间不允许有别的语句；
- 同一般函数一样，函数模板也可以重载；
- 函数模板与同名的非模板函数可以重载。调用的顺序是：首先寻找一个参数完全匹配的非模板函数，如果找到了就调用它；若没有找到，则寻找函数模板，将其实例化，产生一个匹配的模板函数，若找到了，就调用它。
