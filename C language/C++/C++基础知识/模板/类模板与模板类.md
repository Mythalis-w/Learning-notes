- **类模板**：建立一个通用类，其数据成员、成员函数的返回类型和形参类型不具体指定，用一个虚拟的类型来代表。

- **模板类**：使用类模板定义对象时，系统会根据实参的类型来取代类模板中虚拟类型从而实现不同的模板类。

## 类模板的声明：
```cpp
template <typename 类型参数>
class 类名
{
    类成员声明;
};
```

举个例子

```Cpp
#include <iostream>
using namespace std;

template <typename AT>
class Compare
{
    AT x, y; // 两个待比较的成员变量，类型由模板参数AT决定
public:
    // 构造函数，初始化x和y
    Compare(AT a, AT b) {
        x = a;
        y = b;
    }
    // 比较函数，返回较大值
    AT max() {
        return (x > y) ? x : y;
    }
};

int main() { // 标准main写法，原代码的void main()是早期非标准写法
    // 实例化模板类，指定不同的类型参数
    Compare<int> com1(3, 7);
    Compare<double> com2(12.34, 56.78);
    Compare<char> com3('a', 'x');

    // 调用max()函数并输出结果
    cout << "Compare<int> com1(3,7)其中的最大值是：" << com1.max() << endl;
    cout << "Compare<double> com2(12.34,56.78)其中的最大值是：" << com2.max() << endl;
    cout << "Compare<char> com3('a','x')其中的最大值是：" << com3.max() << endl;

    return 0;
}
```
输出结果是
```plaintext
Compare<int> com1(3,7)其中的最大值是：7 
Compare<double> com2(12.34,56.78)其中的最大值是：56.78 
Compare<char> com3('a','x')其中的最大值是：x
```
