# 0.后期编写程序

后期我们要编写一些程序，这是以编译器为Clion进行讲解
```text
D:/myproject/
├── include/  # 这里用来放类声明（头文件）
│   └── Student.h  
├── src/  # 这里来放类定义
│   └── Student.cpp
└── main.cpp  # 这里是主程序（主程序也可以放在src中）
```

Clion的编译要求.h和.cpp文件放在一个项目里，正常不在项目里是找不到的，开始他会把所有的cpp文件编译变成obj文件，这个过程会调用.h文件，然后连接器把所有的obj文件链接在一起，通过查找类的名字对应来寻找类的定义

# 1.头文件（.h）怎么写

一般头文件是用来声明的，头文件的引用只和文件名字有关，和文件里面的东西无关
```C++
#ifndef HEADER_NAME_H 
#define HEADER_NAME_H 

// 头文件内容

#endif
```
