	
# 1.类和对象

在C++中，允许使用class和struct两种方法去定义类，其中，class默认所有的类都是私有的，struct默认所有的类都是共有的，除此以外，两者完全等价。

```C++
class Student {
// 默认是 private 区域
    std::string private_name;  // 这个是私有的，类外不能访问

public:  // 从这里开始，后面的成员都是公有的
    std::string name;  // 公有的
    int age;           // 公有的

    void display() {   // 公有的成员函数
        // 在类的内部，可以访问私有成员
        private_name = "内部可访问"; 
    }

private: // 又可以切换回私有区域
    double secret_score; // 又是私有的了
};
```
类访问其中的共有属性或者共有的成员时，使用`class_name.name`

类类型也有指针变量
```C++
student stu;
student *p = &stu;
(*p).name = "Liping"
student student[3];
p = student+2;
```
指针指向动态分配的内存也是可以的
```C++
student *p = new student;
p -> name = "Wangping";
delete p;
p= new student[3];
```

在类中，成员函数可以像成员那样被调用，如`stu.print()`
成员函数可以在类的内部被定义，也可以在类的外部被定义，如果在类的外部被定义，需要加上作用域说明符

```C++
struct student{
	string name;
	double grade;
	void print(); // 函数声明
```

## 构造函数

这一点和python中的__init__很像，每一个类在创建的时候都会自己生成并调用一个构造函数，我们上面的类并没有实现将参数传入程序的作用，构造函数就可以实现传入参数的作用

例子如下
```C++
#include<iostream>  
#include<string>  
using namespace std;  
  
class student{  
private:  
    string name;  
    int id;  
    double score;  
public:  
    student(string s,int i ,double n){  
        name = s; id = i; score =n;  
    }  
    void print(){  
        cout << name <<" "<< id<<" " << score<<endl;  
    }  
};  
  
int main(){  
    string name;  
    int id;  
    double score;  
  
    cout << "name " << "id " << "score";  
    cin >> name >> id >> score;  
  
    student stu(name , id ,score);  
    stu.print();  
    return 0;  
}
```

==如果你想写构造函数，但是只是想写出来，可以`name(){;}`，和大部分类型一样，不想写东西就写一个；== 

除了构造函数可以传入参数，类还可以通过初始化列表来赋值,但是这种初始化顺序和声明顺序必须一样，也叫做：函数参数的默认值

```C++
class Example {
    std::string s;
    int i;
public:
    // 低效方式：s先被默认构造为空字符串，然后被赋值为"Hello"
    Example() {
        s = "Hello"; 
        i = 10;
    }

    // 高效方式：s直接通过拷贝构造函数初始化为"Hello"
    Example() : s("Hello"), i(10) {
    }
};
```
# 2.内联函数

内联函数是C++中的一个特性，用关键字 `inline` 修饰，目的是**减少函数调用的开销**。

- 普通函数调用：需要压栈、跳转、返回等操作
- 内联函数：在==**编译**时**将函数体直接插入**到调用处==，像宏一样展开，但保留了函数的语法特性

```C++
// 普通函数
int add(int a, int b) {
    return a + b;
}
int result = add(3, 5);  // 调用函数，有开销

// 内联函数
inline int add(int a, int b) {
    return a + b;
}
int result = add(3, 5);  // 编译时直接替换为 3 + 5
```

可以消除一部分函数调用的开销
==注意：类内定义的函数通常是默认内联的 ; 而复杂的函数哪怕写了inline也可能不会内联，这个看编译器了,inline修饰只是给编译器建议是否内联，而不是强制。==

# 3. 析构函数

析构函数是C++中一个特殊的成员函数，在对象**被销毁时自动调用**，用于执行清理工作。

- 特性：
- **无参数**，无返回值
    
- **不能重载**（每个类只有一个析构函数）
    
- 对象销毁时**自动调用**
    
- 主要用途：**释放资源**（堆内存、文件句柄、网络连接等）

# 4. 友元函数

**友元函数**（Friend Function）是 C++ 中定义在类的外部，但有权访问类的所有私有（private）成员和保护（protected）成员的普通函数。

其特点就是在类内被声明，但是定义要写在外面。类的内部，使用关键字 `friend` 加上函数的原型进行声明，可以放在类的私有和公有部分，都是可以的。在函数外部定义

```C++
#include<iostream>
using namespace std;

class student{
    int id;
    double grade;
public:
    void setData(int i, double s){
        id = i;
        grade = s;
    }
    friend double average(student *p, int count);
};

double average(student *p, int count){
    if(count == 0) return 0;
    double sum = 0;
    for(int i = 0; i < count; i++){
        sum += (*(p+i)).grade;
    }
    return sum / count;
}

int main() {
    student stu[101];
    int id;
    double score;
    int count = 0;
    
    while (true) {
        cin >> id;
        if (id == 0) break;
        
        cin >> score;
        stu[count].setData(id, score);
        count++;
    }
    
    double avg = average(stu, count);
    cout << avg << endl;
    
    return 0;
}
```

# 5. 类的继承（派生类）

我们首先讲解protected访问修饰符：`protected` 是C++类中的**访问控制修饰符**，它的访问权限介于 `public` 和 `private` 之间。

### 访问权限对比
| 访问修饰符         | 本类内部  | 派生类内部  | 外部（其他函数/对象） |
| ------------- | ----- | ------ | ----------- |
| **public**    | ✅ 可访问 | ✅ 可访问  | ✅ 可访问       |
| **protected** | ✅ 可访问 | ✅ 可访问  | ❌ 不可访问      |
| **private**   | ✅ 可访问 | ❌ 不可访问 | ❌ 不可访问      |

讲完了protected，可以讲派生类了，派生类就是类的继承
语法：
```C++
class 基类 {
    // 基类成员
};

class 派生类 : 访问说明符 基类 {
    // 派生类特有的成员
};
```
访问说明符有三种，即private public protected ，举个例子，T 继承 就是在基类中public和protected中的所有东西都能被继承，但是所有继承的东西在派生类中都会变成T
示例：

```C++
#include <iostream>
using namespace std;

class Animal {
public:
    void eat() { cout << "Eating" << endl; }
    
protected:
    void sleep() { cout << "Sleeping" << endl; }
    
private:
    void breathe() { cout << "Breathing" << endl; }  // 永远无法被继承
};

// 默认 private 继承
class Cat : Animal {  // 相当于 private Animal
public:
    void catAction() {
        eat();      // ✅ 可以（变成 private 成员）
        sleep();    // ✅ 可以（变成 private 成员）
        // breathe(); // ❌ 永远不能访问 private 成员
    }
};

// 显式 public 继承（通常想要的方式）
class Dog : public Animal {
public:
    void dogAction() {
        eat();      // ✅ 可以（保持 public）
        sleep();    // ✅ 可以（保持 protected）
    }
};

int main() {
    Cat cat;
    Dog dog;
    
    // Cat 是 private 继承
    // cat.eat();  // ❌ 错误！eat 在 Cat 中是 private
    
    // Dog 是 public 继承
    dog.eat();      // ✅ 可以！eat 保持 public
    
    // dog.sleep();  // ❌ 错误！sleep 是 protected，外部不能访问
    
    return 0;
}
```

# 6. this指针

this指针是一个**隐含的指针**，它指向**当前对象本身**。当你在类的成员函数中访问成员变量或其他成员函数时，this指针就在背后起作用。

```C++
class Person {
private:
    string name;
    int age;
    
public:
    Person(string name, int age) {
        // 使用this指针区分参数和成员变量
        this->name = name;  // 左边的name是成员变量，右边的name是参数
        this->age = age;     // this->age表示当前对象的age
    }
    
    void introduce() {
        // 这里实际上隐含了this指针
        cout << "我叫" << this->name << "，今年" << this->age << "岁" << endl;
        // 等价于：cout << "我叫" << name << "，今年" << age << "岁" << endl;
    }
    
    // 返回当前对象的引用
    Person& growOlder() {
        this->age++;
        return *this;  // 返回当前对象本身
    }
};
```

# 7.静态成员

在 C++ 中，静态成员是属于**整个类**而非类的某个对象的成员，分为**静态成员变量**和**静态成员函数**。可以把它理解为：类的 “共享资源”，所有对象共用这一份数据 / 方法，不随对象创建 / 销毁而变化。

下面举一个静态成员变量的例子
```C++
#include <iostream>
using namespace std;

class Student {
public:
    string name;          // 普通成员变量（每个对象独有）
    static int totalNum;  // 静态成员变量（所有学生共享的总人数）
    // 构造函数
    Student(string n) : name(n) {
        totalNum++;  // 每创建一个对象，总人数+1
    }
};

// 静态成员变量类外初始化（语法：类型 类名::变量名 = 初始值） 类外定义
int Student::totalNum = 0;  

int main() {
    Student s1("张三");
    Student s2("李四");
    
    // 两种访问方式：对象访问 / 类名直接访问（推荐）
    cout << "通过对象访问：" << s1.totalNum << endl;  // 输出2
    cout << "通过类名访问：" << Student::totalNum << endl;  // 输出2
    
    // 修改静态变量，所有对象共享
    Student::totalNum = 5;  //对象名.静态变量也是可以访问的
    cout << s2.totalNum << endl;  // 输出5
    return 0;
}
```

## 静态成员函数：
- 属于类而非对象，没有隐含的`this`指针； （因此无法访问普通成员）
- 只能访问**静态成员变量 / 静态成员函数**，不能访问普通成员（因为普通成员依赖具体对象）；
- 可以通过类名直接调用，无需创建对象。

```C++
#include <iostream>
using namespace std;

class Calculator {
public:
    static int sum;  // 静态成员变量：累计求和结果
    // 静态成员函数：计算两数之和并累计
    static int add(int a, int b) {
        sum += (a + b);  // 只能访问静态成员sum
        return a + b;
    }
};

int Calculator::sum = 0;  // 初始化静态变量

int main() {
    // 无需创建对象，直接通过类名调用静态函数
    cout << "1+2=" << Calculator::add(1, 2) << endl;  // 输出3
    cout << "累计和：" << Calculator::sum << endl;     // 输出3
    
    Calculator::add(3, 4);
    cout << "累计和：" << Calculator::sum << endl;     // 输出10
    return 0;
}
```

这两种方法一般在统计类的对象的时候去使用.
