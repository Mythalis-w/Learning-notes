# 局部变量与全局变量

如果外部定义一个变量，那么在一个没有传入这个变量的函数里是不存在的，如下

```C
#include <stdio.h>
void example(){
	j=90;	
}

int main(){
	int j=100;
	example();
	printf("%d",j);
	return 0;
}
```

这时候如果j能输出则依旧是100，而不是90，除非声明global ，并且这时候会报错，因为这个函数不知道j是多少。

```C
#include <stdio.h>
int j;

void example(){
	j=90;	
}

int main(){
	int j=100;
	example();
	printf("%d",j);
	return 0;
}
```

在前面声明变量，就是全局变量，这时候输出的就是90，且不会报错。

如果传入的是一个数组，那么修改函数里面的arr，外部的arr也会一起修改，因为这时候函数传递的是指针，直接指向的地址，除非传入的时候加上 const char arr ，那这时候arr就不会改变
