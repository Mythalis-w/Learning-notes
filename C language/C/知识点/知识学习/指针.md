 ```C
 void bubbleSort(int arr[], int n) {
    int temp, sign;
    for(int i = 0; i < n - 1; i++) {
        sign = 0;
        for(int k = 0; k < n - 1 - i; k++) {
            if(arr[k] > arr[k + 1]) {
                temp = arr[k];
                arr[k] = arr[k + 1];
                arr[k + 1] = temp;
                sign = 1;
            }
        }
        if(sign == 0) break;
    }
}
 ```

这是一个冒泡排序，如果输入(arr+3, 7)，==arr+3 等价于 &arr\[3]==， 它是指针运算的，从索引为3的元素作为首地址，排7个地址。

同理的，gets(s+2) == gets(&s\[2])，puts也一样，如下例子

# 指针传递

```C
#include <stdio.h>

// 正确！传递指针（地址）
void swap_correct(int *pa, int *pb) {
    int temp = *pa;   // *pa 取出 pa 指向的值（即 x 的值）
    *pa = *pb;        // 将 pb 指向的值赋给 pa 指向的位置
    *pb = temp;       // 将临时值赋给 pb 指向的位置
}

int main() {
    int x = 10, y = 20;
    
    printf("交换前：x=%d, y=%d\n", x, y);
    swap_correct(&x, &y);  // 传递 x 和 y 的地址
    printf("交换后：x=%d, y=%d\n", x, y);  // 成功交换！
    
    return 0;
}
```

## & and *

&是取地址运算符，* 是取值运算符（注意，在声明和使用时，\*代表了不同的意思，声明的时候仅仅是声明他是一个指针，但是使用的时候则表示取值运算）

如
```C
#include<stdio.h>
int main(){
	int a,b,*ptr1,*ptr2;
	a=2;
	b=3;
	ptr1=&a;
	ptr2=&b;
	*ptr1 = *ptr2;  // 把b的值赋给a，ptr1 != ptr2 ，但是a=b
	
	a=3;
	b=5;
	ptr1 = ptr2 // 把b的地址赋值给ptr1,因此，*ptr1=*ptr2=b
	
}
```

即字面上我们可以视作ptr == &arr\[0]   , \*ptr == arr\[0]

注意：数组在用指针的时候就已经传递了初始地址

```C
int a[1000]
int *ptr = a  //即，ptr = &a[0] = a
```

等价于
```C
int a[1000]
int *ptr
ptr = a  //是通过a+i = &a[i]来给地址的 
```

一般数组的传递就三种方法，下标法，地址法（传统的arr\[1]）（a+i = &a\[i]），指针法（通过ptr+a 来更改指针的位置）

我们假设ptr是arr这个数组里的中间的某一个地址，他的printf和%p或者%s有关
如%s，从ptr这一个地址的元素一直处理到最后一个元素
%p，直接输出ptr这个元素的地址
即ptr存储的是指针（某个地址），至于是怎么处理依据函数来定义。

# 1. 数组指针和指针数组

## 指针数组

它的本质是数组，里面的元素是指针，如 `int* ptr_arr[100]` 它就可以存储100个地址，但是只能存储int类型的指针，如果想存储多类型的就应该用 `void* ptr_arr[100]` 

指针数组可以去拿来构造字符串数组，如`char *arr[3] = {"a", "bb", "ccc"};` ，这时候，这个数组存入的是这些字符串的首字母的地址，使用方法如下

```C
1. arr：指针数组首地址
    
2. arr + 1：移动到第二个指针的地址
    
3. *(arr + 1)：解引用得到 `arr[1]`（即字符串"bb"的地址）
    
4. *(arr + 1) + 1：在该字符串地址上加1（指向第二个字符'b'）
    
5. *(*(arr + 1) + 1)：解引用得到字符'b'   //等价于arr[1][1],尽管是一维，也可以这样用
```

对于常见的字符串的函数，其需要输入的大部分是`char*`类型，而一个二维数组的类型是`char(*)[10]`，而指针数组的名称的类型也是`char **  // 这是一个指向char*的指针（二级指针）` 因此在使用时大部分需要用`* _name_`来使用。

## 数组指针

- 他的本质是指针，指向一个数组，即通过指针调用数组，如 `int *p=str` ，p这个指针指向str这个数组。
- 对于二维数组 `int arr[k][N]`，我们可以定义一个**行指针**(二维数组退化成行指针) `int (*p)[N] = arr`。此时 `p` 指向的是**第一行**（一个包含N个元素的数组）。通过对 `p` 解引用（`*p`）得到的是指向第一行第一个元素的**列指针**（如果解引用是`*(p+1)+2` ,则得到的是指向第二行第三个元素的列指针）。

如果有一个函数`void Swap(int *x, int *y)` 此时，x和y传入的是一个指针(int*) 类型，也就是说，需要传入地址（这个如果传入数组的话，int \*arr 和int arr\[] 等价）

```C
int a, b;
scanf("%d %d", &a, &b);
Swap(&a,&b)  //这个就是非常典型的传入地址
```

```C
int a = 32, b = 90;
int *x = &a;   // x指向a，x储存的是a的地址
int *y = &b;   // y指向b，y储存的是b的地址
Swap(x, y);    // 传递指针变量（里面存储的是地址）
```

函数输入的是一个地址，我们在调用这个值的时候也需要用\*去解引用，如下，去分割一个数字的整数和小数部分

```C
void splitfloat( float x, int *intpart, float *fracpart ){
```

# 3. 二维数组与二级指针

**二维数组名退化为指向一维数组的指针**  ，即一个数组指针 (`int (*)[N]`)，因此二维数组名是一个行指针，每一次解引用(**解引用就是使用 `*` 运算符来获取指针指向的值。**)都是降级

```C
int arr[1000][1000];
*arr // 对指向一维数组的指针解引用，最后的结果是指向第一行第一列元素的指针 &arr[0][0]或arr[0]
**arr // 对指向第一行第一列元素的指针解引用，结果是第一行第一列的元素  arr[0][0]

*(arr+1) // 对指向第二行的指针解引用，最后的结果是指向第二行第一列的指针 &arr[1][0]/arr[1]
**(arr+1) // 对指向第二行第一列元素的指针解引用，结果是第二行第一列元素  arr[1][0]

*(arr+1)+2 // 对指向第二行的指针解引用，然后指针后移2,指向第二行第三列元素的指针  &arr[1][2]
*(*(arr+1)+2) // 对指向第二行第三列元素的指针解引用，得到这个值  arr[1][2]
```

**二级指针**和二维数组不一样（上面的所有都不是二级指针）

```C
int arr[3][4];          // 二维数组

// 定义指向"行指针"的指针
int (*p1)[4] = arr;     // p1指向第一行
int (**pp1)[4] = &p1;   // pp1(二级指针)指向p1

// 定义指向"列指针"的指针  
int* p2 = *arr;         // p2指向第一行第一列
int** pp2 = &p2;        // pp2(二级指针)指向p2
```

# 二维数组的访问与储存
==注意，在多维数组的时候，心里务必从第零行第零列开始计算，但是输出的时候，我们将第零行叫做第一行==

 
1. 下标法

```C
char str[1000][1000];
printf("%s",str[1]);  //会输出第二行的字符串
printf("%c",str[1][1]);  //会输出第二行第二列的元素
```

==注意，%s 和 puts 要输出字符串所需要的只是第一个字符的地址==
2. 地址法

```C
int arr[1000][1000];
printf("%d",*(arr+1))  //会输出第二行的所有数字
printf("%d",*(arr+1)+2)  //会输出第二行第三列的数字
```









通过下标法直接访问是容易理解的一个方式，下面我们讲解下储存，比如输入`hello world`吧，怎么分别储存在每一行中，一种比较方便的方式就是通过strtok

```C
#include <stdio.h>
#include <string.h>
int main(){
char temp[1000];
int row=0;
char str[1000][1000];
gets(temp);
char* token =strtok(temp, " ");
while(token != NULL){
	strcpy(str[row], token);
	row++;
	token = strtok(NULL, " ");
	}
return 0;
}
```
