```input
Hello World!
2
```

```output
llo World!He
```


代码区：
```C
#include<stdio.h>
#include<string.h>
int main(){
    char str[101];
    gets(str);
    int n;
    scanf("%d",&n);
    if(n>strlen(str)){
        n=n-strlen(str);
    }
    for(int i=n;i<strlen(str);i++){
        putchar(str[i]);
    }
    for(int i=0;i<=n-1;i++){
        putchar(str[i]);
    }
    return 0;
}
```

- strlen(name) 是得到字符串的长度
- putchar(name\[i])则可以输出单个字符
- strlen(name) 可以获取字符长度

gets和scanf的区别，gets在输入完后会自动添加\0，变为字符串，scanf("%s",……)也会自己添加\0来变成字符串，如果用for循环一个字符一个字符的读取，且最后不输入\0那么输入的就不是字符串而是一个一个字符。

# 1. 长度与比较
## 1. strcmp 的用法

strcmp是用来比较字符串的内容的，且是根据ASCII来进行比较
使用方法如下 `int strcmp(char *str1,  char *str2);

```C
#include <stdio.h>
#include <string.h>
int main(){
	if (strcmp(str1, str2) == 0) {
	    // 两个字符串完全相等
	}
	if (strcmp(str1, str2) < 0) {
	    // str1 小于 str2（按ASCII顺序）
	}
	if (strcmp(str1, str2) > 0) {
	    // str1 大于 str2（按ASCII顺序）
	}
}
```

注意strcmp不会全部比较，而是逐个比较，一但发现不一样就会返回不一样的字符ASCII的差值（注意：差值是前减后），然后终止运行。（注意：str1和str2可以直接输入字符串，用双引号括起来即可）

### 扩展

比较中间字符的方法，即通过指针偏移，如下

```C
#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Hello World!";
    
    // 从第6个字符开始比较（跳过 "Hello "）
    char *start1 = str + 6;  // 指向 "World!"
    
    char str2[] = "World of C";
    
    // 比较前5个字符
    int result = strncmp(start1, str2, 5);
    
    if (result == 0) {
        printf("从中间开始比较: 相同\n");
    } else {
        printf("从中间开始比较: 不同\n");
    }
    
    return 0;
}
```

当然，结果一定是相同。

# 3. 查找与分割

## 1.strchr 和 strrchr的用法

总用法都是  `_name_(str, z)` ，其中z如果是一个字符，需要带 ‘ ’ ，z也可以是一个ASCII值

- strchr是用来查找某一个字符第一次出现的位置，`strchr(str, 'c')` 
- strrchr是用来查找某一个字符最后一次出现的位置的，`strrchr(str, 'c')

注意strchr和strrchr 都是返回一个指针，我们也要用一个已经定义的指针去接受它

```C
char str[] = "Hello World";
```

## 2.strstr

这个函数是用来查找子串的，用法`strstr(const char *haystack, const char *needle);` 
- **haystack**：要搜索的目标字符串（主字符串）
- **needle**：要查找的子字符串
这个函数依旧返回一个指针，所以我们要用一个指针是接受，指向的子串第一个字符的地址；如果没找到的，返回NULL；如果子串是空的，则返回第一个字符的地址

```C
#include <stdio.h>
#include <string.h>

int main() {
    const char *str = "Hello, World! Welcome to C programming.";
    const char *substr = "World";
    
    char *result = strstr(str, substr);
    
    if (result != NULL) {
        printf("子字符串在位置: %ld\n", result - str);
    } else {
        printf("未找到子字符串\n");
    }
    
    return 0;
}
```

## 3.strtok

`strtok` 是 C 语言中用于**分割字符串**的函数，用法`char *strtok(char *str, const char *delim)` 
- **str**：要分割的字符串（第一次调用时传入）
- **delim**：分隔符字符串（可以包含多个分隔字符）==一定是字符串，字符不行==
注意：strtok会修改原字符串，即原字符串会被修改为第一个分割的字符；然后使用的时候，第一次调用输入str，后续想继续分割的话需输入NULL，这样才会继续修改原来的字符串，返回依旧是一个指针，例子如下：

```C
#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "apple,banana,orange,grape";
    const char *delim = ",";
    
    // 第一次调用，传入原字符串
    char *token = strtok(str, delim);
    
    while (token != NULL) {
        printf("Token: %s\n", token);
        // 后续调用，传入 NULL
        token = strtok(NULL, delim);
    }
    
    // 注意：原字符串已被修改
    printf("原字符串现在: %s\n", str);  // 输出: apple
    
    return 0;
}
```

输出

```output
Token: apple
Token: banana
Token: orange
Token: grape
原字符串现在: apple
```

注意，strtok中支持多个分隔符，如`" -!,."` 代表着在 空格、逗号、点、连字符、感叹号  处分割，且分割的符号会自己抛弃掉，如果中间有多个空格，也都会直接抛弃掉
## 4.strpbrk

这个函数相当于strchr的扩展版本，可以查找多个字符在某个字符中的位置
用法`char *strpbrk(const char *str1, const char *str2);` 
其中str2可以是一个字符串，str1是要搜索的字符串，返回依旧是一个指针，指向找到的第一个字符的地址
基本用法如下

```C
#include <stdio.h>
#include <string.h>

int main() {
    const char *text = "Hello, World! 123";
    const char *digits = "0123456789";
    
    // 在 text 中查找第一个数字
    char *result = strpbrk(text, digits);
    
    if (result != NULL) {
        printf("找到数字: %c\n", *result);        // 输出: 1
        printf("从该位置开始: %s\n", result);     // 输出: 123
        printf("位置索引: %ld\n", result - text); // 输出: 14
    } else {
        printf("未找到数字\n");
    }
    
    return 0;
}
```
