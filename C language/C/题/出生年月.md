以上是新浪微博中一奇葩贴：“我出生于1988年，直到25岁才遇到4个数字都不相同的年份。”也就是说，直到2013年才达到“4个数字都不相同”的要求。本题请你根据要求，自动填充“我出生于y年，直到x岁才遇到n个数字都不相同的年份”这句话。 输入格式： 输入在一行中给出出生年份y和目标年份中不同数字的个数n，其中y在[1, 3000]之间，n可以是2、或3、或4。注意不足4位的年份要在前面补零，例如公元1年被认为是0001年，有2个不同的数字0和1。 输出格式： 根据输入，输出x和能达到要求的年份。数字间以1个空格分隔，行首尾不得有多余空格。年份要按4位输出。注意：所谓“n个数字都不相同”是指不同的数字正好是n个。如“2013”被视为满足“4位数字都不同”的条件，但不被视为满足2位或3位数字不同的条件。 输入样例1： 1988 4 输出样例1： 25 2013 输入样例2： 1 2 输出样例2： 0 0001

好的，我们来分析并解决这个问题。

## 问题分析

1. **输入**：出生年份 `y` 和不同数字个数 `n`
    
2. **目标**：从出生年份开始，找到第一个满足“年份的4位数字中恰好有 `n` 个不同数字”的年份
    
3. **输出**：
    
    - `x`：找到该年份时的年龄（年份差）
        
    - 目标年份（按4位输出，不足补0）
        
4. **注意**：
    
    - 年份要格式化为4位，不足补前导0
        
    - “n个数字都不相同”是指**恰好有n个不同的数字**，不是“至少n个”
        
    - 例如：`2013` 恰好有4个不同数字，满足 `n=4`，但不满足 `n=2` 或 `n=3`
        
    - 年龄从0岁开始计算（出生当年0岁）
        

## 解题思路

1. **年份格式化**：将年份转为4位字符串（不足4位补0）
    
2. **检查函数**：判断一个年份字符串中不同数字的个数
    
3. **循环查找**：从出生年份开始逐年检查，直到找到满足条件的年份
    
4. **计算年龄**：目标年份 - 出生年份

#include <stdio.h>
#include <string.h>

// 判断字符串中不同数字的个数
int count_distinct_digits(char *year_str) {
    int digit_count[10] = {0};  // 记录0-9每个数字出现的次数
    int distinct = 0;
    
    // 统计每个数字出现的次数
    for (int i = 0; i < 4; i++) {
        digit_count[year_str[i] - '0']++;
    }
    
    // 统计出现过的不同数字个数
    for (int i = 0; i < 10; i++) {
        if (digit_count[i] > 0) {
            distinct++;
        }
    }
    
    return distinct;
}

int main() {
    int y, n;
    scanf("%d %d", &y, &n);
    
    int current_year = y;
    char year_str[5];  // 4位年份 + '\0'
    
    while (1) {
        // 将年份格式化为4位字符串
        sprintf(year_str, "%04d", current_year);
        
        // 检查不同数字个数
        if (count_distinct_digits(year_str) == n) {
            // 找到满足条件的年份
            printf("%d %s\n", current_year - y, year_str);
            break;
        }
        
        current_year++;
    }
    
    return 0;
}
