输入一个字符串，对该字符串进行逆序，输出逆序后的字符串。

### 输入格式：

输入在一行中给出一个不超过80个字符长度的、以回车结束的非空字符串。

### 输出格式：

在一行中输出逆序后的字符串。

### 输入样例：

```in
Hello World!
```

### 输出样例：

```out
!dlroW olleH
```

答案

```C
#include<stdio.h>
int main(){
    char arr[81];
    int i=0;
    char c;
    while((c=getchar()) != '\n'){
        arr[i]=c;
        i++;
    }
    for(int j=i-1;j>=0;j--){
        printf("%c",arr[j]);
    }
    return 0;
    
}
```
