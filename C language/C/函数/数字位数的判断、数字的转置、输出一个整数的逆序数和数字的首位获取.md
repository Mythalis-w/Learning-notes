```C
int Count(int n){
    int count=0;
    while(1){
        if(n!=0){
            count++;
            n=n/10;
        }
        else return count;
    }
}
```

通过上述代码，可以实现位数的判断，if中的条件是(n!=0)而不是(n%10!=0)，这种写法可以避免n的最后一位是0的情况


```C
int reverseNumber(int n) {
    if (n == 0) return 0;
    int sign = 1;  //sign是用来判断正负的
    if (n < 0) {
        sign = -1;
        n = -n;
    }
    int reversed = 0;
    while (n != 0) {
        reversed = reversed * 10 + (n % 10);
        n /= 10;
    }
    return reversed * sign;
}
```

输出一个整数的逆序数

```C
int reverse(int number) {
    int result = 0;
    int sign = 1;  
    if (number < 0) {
        sign = -1;
        number = -number;  
    }
    while (number > 0) {
        result = result * 10 + number % 10;  
        number /= 10; 
    }
    return result * sign; 
}
```
